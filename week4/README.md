# Week 4: Edge Detection (การตรวจจับขอบ)

## Overview
บทเรียนสัปดาห์ที่ 4 ครอบคลุมเทคนิคการตรวจจับขอบ (Edge Detection) ซึ่งเป็นหนึ่งในเทคนิคพื้นฐานของการประมวลผลภาพและการมองเห็นของคอมพิวเตอร์

## Topics Covered

### 1. บทนำการตรวจจับขอบ
- **ความเข้าใจเบื้องต้น:** ขอบคือการเปลี่ยนแปลงเฉพาะที่ที่สำคัญในความเข้มของภาพ
- **ความสำคัญ:** มีประโยชน์ในการวิเคราะห์ภาพ การตรวจจับรูปร่าง การจดจำวัตถุ และการแบ่งส่วนภาพ

### 2. แนวคิดพื้นฐาน
- **Gradient (ความชัน):** การวัดการเปลี่ยนแปลงความสว่างในชุดของพิกเซล
- เป็นเวกเตอร์ที่ประกอบด้วยทิศทางและขนาดของการเปลี่ยนแปลง

### 3. ตัวดำเนินการตรวจจับขอบ

#### Roberts Operator
- ใช้ mask ขนาด 2×2
- **ข้อดี:** เร็วที่สุด ตรวจจับขอบทแยงได้ดี
- **ข้อเสีย:** ไวต่อสัญญาณรบกวน ผลลัพธ์หยาบ
- **เหมาะสำหรับ:** ภาพที่มีคุณภาพดี

#### Prewitt Operator
- ใช้ mask ขนาด 3×3
- **ข้อดี:** สมดุล มีความเสถียร ต่ำต่อสัญญาณรบกวน
- **เหมาะสำหรับ:** งานทั่วไป

#### Sobel Operator
- ใช้ mask ขนาด 3×3
- **ข้อดี:** ผลลัพธ์ดี ประเมินทั้ง x และ y gradient
- **ข้อเสีย:** ประมวลผลมากกว่า
- **เหมาะสำหรับ:** งานทั่วไป (ได้รับความนิยม)

#### Canny Edge Detector
- อัลกอริทึมแบบหลายขั้นตอน
- สามารถตรวจจับขอบที่หลากหลายในภาพ

### 4. ตารางเปรียบเทียบตัวดำเนินการ

| คุณสมบัติ | Roberts | Prewitt | Sobel |
|---------|---------|---------|-------|
| ขนาดเคอร์เนล | 2×2 | 3×3 | 3×3 |
| ความเร็ว | เร็วที่สุด ⚡ | ปานกลาง | ปานกลาง |
| ความอ่อนไหว | สูงต่อสัญญาณรบกวน | ต่ำกว่า Roberts | ต่ำกว่า Roberts |
| ขอบทแยง | ดีที่สุด | ปานกลาง | ปานกลาง |
| ประสิทธิภาพ | ต่ำ | สูง | สูงที่สุด |

## วิธีการ (Algorithm Steps)

1. **เลือก Operator** → ได้รับ filter ที่เหมาะสม
2. **คำนวณ Gradient Components:** หา Gx และ Gy
3. **คำนวณ Edge Magnitude:**
   - เพิ่มกำลังสอง: Gx² + Gy²
   - ถอดรูท: √(Gx² + Gy²)
4. **ได้ผลลัพธ์:** เส้นขอบจะแสดงเป็นสีขาว ส่วนที่เหลือเป็นสีดำ
